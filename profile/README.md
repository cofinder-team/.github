# Macguider: 데이터 구조화 기반 중고 애플 제품 종합 비교 서비스

여러 플랫폼에 흩어져 있는 중고 애플 제품을 한 곳에 모아 편리하게 제품들을 검색, 비교할 수 있는 종합 중고 제품 비교 서비스로,

상태와 사양별로 시세를 제공하고 업자 및 사기 의심 매물을 필터링하여 거래 전에 필요한 중고 제품 리서치 과정을 최소화하고자 합니다.
